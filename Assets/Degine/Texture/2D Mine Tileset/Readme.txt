======================================================
2D Mine Tileset is a package of high quality tiled textures of environment and items to help you create levels for platformers, shooter or even endless runners.

======================================================
Features
======================================================
 - 192 environment tiles in 3 color themes
 - 16 items like spikes, treasures and dynamite
 - Each tile is 128*128 in PGN format with alpha
 - PSD source files are included to let you create even more varieties
 - Example scene included
